EMI Calculator

This is a simple web-based EMI (Equated Monthly Installment) calculator application built using HTML, CSS, and JavaScript.

Features:

Loan EMI Calculation: Calculates the Equated Monthly Installment (EMI) based on the loan amount, interest rate, and tenure provided by the user.
Total Interest: Displays the total interest payable over the loan tenure.
Total Amount: Shows the total amount payable including the principal loan amount and interest.
Interactive Chart: Visualizes the breakdown of total interest and principal loan amount in a pie chart.

How to Use:

1. Enter the loan amount, interest rate, and tenure (in months) into the respective input fields.
2. Click on the "Calculate EMI" button to generate the EMI, total interest, and total amount payable.
3. The results will be displayed on the right side of the screen along with an interactive pie chart illustrating the distribution of total interest and principal loan amount.

File Structure:

- index.html: Contains the structure and content of the web page.
- web.css: Defines the styles and layout of the EMI calculator.

Technologies Used:

HTML: Markup language for structuring the web page.
CSS: Stylesheet language for styling the web page elements.
JavaScript: Programming language for adding interactivity and functionality to the application.
Bootstrap: Front-end framework for responsive and mobile-first design.
Chart.js: JavaScript library for creating interactive charts and graphs.

Responsive Design:

- The application is designed to be responsive and mobile-friendly, ensuring optimal user experience across different devices and screen sizes.
- Media queries are used in CSS to adjust the layout and styling for smaller screens.

Development Setup:

1. Clone the repository or download the files.
2. Open the `index.html` file in a web browser to view the EMI calculator application.
3. Modify the HTML, CSS, or JavaScript files as needed to customize or enhance the functionality.
